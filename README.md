# Crest Partners — Next.js Website

## Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Add font files
Copy the three Spartan font files to `public/fonts/`:
- `Spartan-Light.ttf`
- `Spartan-Regular.ttf`
- `Spartan-Bold.ttf`

### 3. Run development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### 4. Build for production
```bash
npm run build
npm start
```

---

## Pages
- `/` — Homepage
- `/services` — Services (Launch · Build · Operate)
- `/case-studies` — Case Studies (with client-side filter)
- `/who-we-are` — Team & founders
- `/why-costa-rica` — Market overview
- `/contact` — Contact form + direct options
- `/careers` — Careers placeholder

## Stack
- Next.js 15 · App Router · TypeScript
- Tailwind CSS v3
- next/font/local (Spartan)
